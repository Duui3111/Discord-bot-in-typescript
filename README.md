## this is a discord bot temlate in tyescript 
# useing the bot 
1. run the ``npm i ``
2. edit the config.ex.ts to config.ts in the src 
3. run ``tsc`` to commile the typescript 
4. do ``node dist/bot.js`` to run the bot
